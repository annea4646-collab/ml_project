# 스마트팜 재배환경 기반 생산량 예측 서비스

## 프로젝트 개요
- 스마트팜 재배환경 데이터와 생육 데이터를 활용하여 생산량을 예측하고 출하 적기를 추천하는 서비스를 개발하는 프로젝트입니다.
- 기상 정보와 가격 데이터를 함께 분석하여 농가의 재배 및 출하 의사결정을 지원하는 것을 목표로 합니다.

## 개발 목표
- 재배환경 및 생육 데이터를 활용한 생산량 예측 모델 구축
- 생육속도 예측 및 예상 수확일 도출
- 기상 정보와 가격 데이터를 활용한 출하 적기 추천 기능 구현
- 기상 특보 기반 긴급 수확 알림 기능 제공
